# BilibiliLiveRecordSync (BLRS)

 此项目是由Spring Boot + Thymeleaf实现的同步系统

用于同步Bilibili直播的直播回放

因直播回放好像只保存两个月，于是想出做出这个项目把直播回放同步到OneDrive

初学Spring Boot

下载部分请看我另一个项目[BLRS](https://github.com/CNawalol/BLRS)
